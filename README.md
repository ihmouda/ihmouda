## 👨‍💻 Open Source Projects

### [JNMentionTextView](https://github.com/ihmouda/JNMentionTextView)
A `UITextView` subclass providing `@mention` annotations for iOS apps. Mentions are treated as atomic units and can be styled independently from surrounding text.

### [JNSegmentedControl](https://github.com/ihmouda/JNSegmentedControl)
An animated, easy-to-use segmented control written in Swift. Supports three segment styles: **text only**, **image only**, and **text + image** (above, below, before, or after).
